## 🎯 Aim Trainer

A browser-based aim training application built with vanilla JavaScript, HTML, and CSS that simulates common FPS aim training scenarios such as flicking, precision clicking, and target tracking.

The project focuses on performance, smooth animations, and efficient hit detection while running entirely in the browser with no external libraries.

## Overview
This aim trainer provides several training modes designed to improve different aspects of mouse control:
- Flick aiming
- Precision clicking
- Target switching
- Continuous tracking
